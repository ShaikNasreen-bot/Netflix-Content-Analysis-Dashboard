# Netflix Content Analysis Dashboard

## Project Overview

This project is an interactive Netflix Content Analysis Dashboard created using Power BI. The dashboard analyzes movies and TV shows available on Netflix and provides insights into ratings, genres, countries, release trends, and content distribution.

---

## Tools Used

* Power BI
* Excel

---

## Dataset Information

The dataset contains 8,000+ Netflix titles including:

* Movie/TV Show Type
* Title
* Director
* Country
* Release Year
* Rating
* Genre
* Duration

---

## Project Workflow

### Step 1: Data Collection

Collected Netflix dataset in CSV/Excel format.

### Step 2: Data Cleaning

Performed data cleaning using Excel and Python Pandas:

* Removed null values
* Fixed column formats
* Handled missing data

### Step 3: Data Transformation

Transformed dataset in Power BI using Power Query Editor.

### Step 4: Dashboard Development

Created interactive visualizations:

* KPI Cards
* Bar Charts
* Donut Charts
* Line Charts
* Slicers and Filters

### Step 5: Insights Generation

Analyzed:

* Content distribution
* Ratings trends
* Country-wise content
* Year-wise growth
* Genre popularity
